El caso de uso define una serie de transacciones aplicables a los diferentes intercambio de datos que se generan durante el proceso de solicitud de una cama para hospitalizar a un paciente, dada una condición específica observada en el/la Paciente durante su atención de urgencia. A continuación se desarrollan los métodos necesarios para generar esas transacciones entre los actores informáticos que intervienen en el proceso.

Transaccion 1: Solicitud de Hospitalización


  • Actor Origen : Sistema de Urgencia
  • Actor Destino : Repositorio HLF
  • Recursos : Bundle, ServiceRequest, Condition, Encounter
  • Operación : Post (Bundle)


La operación debiera ser como sigue: 

POST [base]/


En la estructura del Bundle por cada entry se contiene el recurso a transportar. Debe indicarte el método por cada uno de ellos en la estructura del entry

Transaccion 2: Leer Solicitud de Hospitalización


  • Actor Origen : Repositorio HLF
  • Actor Destino : Gestor de Camas
  • Recursos : ServiceRequest
  • Operación : Get (Bundle)


La operación debiera ser como sigue: 

GET [base]/ServiceRequest/[id]


Sin embargo, y dada la definición de los SearchParametes definidos en el estándar se puede hacer la búsqueda de la solicitud por paciente o fecha de realización de la Solicitud.
Por paciente:

GET [URL_Base]/ServiceRequest?patient=[id_Paciente]


De ser por fecha

GET [URL_Base]/ServiceRequest?authored=[fecha]


Transaccion 3: Cambio de Estado en la Solicitud


https://hl7.org/fhir/fhirpatch.html

  • Actor Origen : Gestor de Camas
  • Actor Destino : Repositorio HLF
  • Recursos : ServiceRequest
  • Operación : Patch Params


La operación debiera ser como sigue: 

PATCH [base]/ServiceRequest/id


Para este tipo de operación se utiliza un recurso FHIR llamado Parameters el cual corresponde a un tipo de recurso que no persiste, y se utiliza indicar la operación a realizar. En particular se ha definido para el caso de uso, el agregar un comentario o cambiar el estado de la solucitud en los parámetros comment y status respectivamente 

{
    {
    "resourceType": "Parameters",
    "parameter": [
        {
            "name": "operation",
            "part": [
                {
                    "name": "type",
                    "valueCode": "add"
                },
                {
                    "name": "path",
                    "valueString": "ServiceRequest"
                },
                {
                    "name": "name",
                    "valueString": "note"
                },
                {
                    "name": "value",
                    "valueAnnotation": {
                        "text": "Comentario Nuevo"
                    }
                }
            ]
        },
        {
            "name": "operation",
            "part": [
                {
                    "name": "type",
                    "valueCode": "replace"
                },
                {
                    "name": "path",
                    "valueString": "ServiceRequest.status"
                },
                
                {
                    "name": "value",
                    "valueCode": "completed"
                }
            ]
        }
    ]
}
